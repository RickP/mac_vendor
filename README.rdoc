== Description

Given a MAC address, lookup the vendor of the interface.

Inspired by the Net-MAC-Vendor perl package.

== Examples

  require 'mac_vendor'

  v = MacVendor.new
  v.lookup '00-11-22-33-44-55'

== Install

  $ gem install mac_vendor
